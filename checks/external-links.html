Steam External Links Policy

What links are allowed, what's prohibited, and common mistakes.

Steam allows external links but has rules. The most common rejection? Broken Discord invites. The most common violation? Links to competing stores.

Allowed Links

✓ Generally Permitted

  • Your official website
  • Discord server (must be active and moderated)
  • Twitter / X
  • YouTube channel
  • Twitch
  • Reddit community
  • Official forums
  • Press kit / media pages
  • Wikis (official or fan-made)

Prohibited Links

✗ Will Get Rejected

  • Epic Games Store
  • GOG
  • Humble Bundle (for game purchase)
  • itch.io
  • Any competing storefront
  • Affiliate or referral links
  • Links that bypass Steam for purchases
  • Patreon (if offering game content)
  • Cryptocurrency / NFT sites

Discord Link Rules

Common rejection: Expired or broken Discord invite links. Always use a permanent invite link, not a temporary one.

  • Use permanent invites — Never-expiring links only
  • Server must be active — Dead servers can cause rejection
  • Must be moderated — Clear rules, active moderation
  • Test the link — Before every submission

Social Media Best Practices

  • Link to accounts you actively use
  • Profiles should be game-related, not personal
  • Avoid accounts that primarily promote non-Steam content
  • Keep links updated if you change handles

What Our Check Looks For

  • Discord link detection (discord.gg, discord.com/invite)
  • Competing storefront URLs
  • Affiliate link patterns
  • Suspicious redirect services
  • Social media link formatting
